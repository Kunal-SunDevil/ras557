Bipedal Robot Using 5-Bar Mechanism

Project Overview

This project focuses on the development of a bipedal robot using a 5-bar linkage mechanism. Inspired by the biomechanics of bird locomotion, the robot mimics natural gait patterns such as walking and hopping. The aim is to optimize the end-effector link length based on distance traveled in the x-direction.

Objectives

  • Analyze and replicate bird-like gaits using foldable mechanisms.
  • Simulate kinematics and dynamics using MuJoCo.
  • Optimize end-effector link length for efficient locomotion.

Methodology

  1. Design a foldable 5-bar mechanism for the bipedal robot.
  2. Conduct simulations in MuJoCo to generate and analyze gait patterns.
  3. Fabricate a robot body using lightweight materials like fiberglass.
  4. Control the robot using servo motors and a microcontroller.

Performance Metrics

Metric Description
Maximum Walking Speed 0.2 m/s
Cost of Transport (CoT) 0.2 (dimensionless)
Joint Position Accuracy 0.01 rad
Payload Capacity 0.1 kg
